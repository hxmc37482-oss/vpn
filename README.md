# MartVPN Bot

Телеграм-бот с оплатой через CryptoBot, выдает VLESS/подписку-ключ после оплаты.
Два тарифа: 24 часа (0.5$) и Навсегда (1$).

## Установка

```bash
pip install -r requirements.txt --break-system-packages
```

## Перед запуском обязательно поправь:

1. **Баннер** — положи картинку в `banner/welcome.jpg`. Если файла нет — бот пришлет текст без фото, не сломается.
2. **Кнопка "Поддержка"** в `bot.py` — сейчас там заглушка `https://t.me/YOUR_SUPPORT_USERNAME`, замени на свой юзернейм, иначе кнопка не будет работать (Telegram не примет невалидный URL).
3. **Тарифы** — меняются в `config.py` в словаре `PLANS`, можно добавить сколько угодно вариантов, добавлять/убирать по своему усмотрению.

## Запуск

```bash
python bot.py
```

## Как это работает

1. `/start` → баннер + меню (Прайс / Как подключиться / Поддержка)
2. «Прайс» → список тарифов с кнопками
3. Юзер выбирает тариф → бот создает счет в CryptoBot → кнопки «Оплатить» и «Я оплатил, проверить»
4. После подтверждения оплаты бот присылает VLESS-ключ

## Важное предостережение

Ключ, который выдает бот, **статичный и общий на всех покупателей** (это одна и та же подписка на VPN-сервис, а не индивидуально создаваемый доступ). Это значит:

- Все оплатившие юзеры фактически шарят один и тот же канал — при большом числе клиентов скорость и стабильность будут падать
- Тариф "Навсегда" за 1$ особенно рискованный — ты обязуешься отдавать доступ бессрочно за разовую копеечную оплату, а нагрузка на канал будет только расти со временем
- Если сервис-провайдер (Ben's VPN) не разрешает перепродажу доступа в своем ToS — есть риск, что аккаунт забанят, и тогда все клиенты разом останутся без доступа, включая тех кто заплатил "навсегда"

Для нормального масштабируемого бизнеса стоит перейти на свой VPS с 3x-ui/Xray, где бот через API создает **отдельный** конфиг под каждого клиента, и тариф "навсегда" будет честным, а не обещанием, которое трудно будет выполнить.
